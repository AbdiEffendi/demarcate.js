About demarcate.js

demarcate.js is a jQuery based editor for Markdown. It lets you edit directly in a page and generate Markdown from the HTML elements. demarcate.js is under development by William Hart for the online publishing service to-textr. It supports many basic Markdown functions, however support for extensions is still limited. The library is offered under the GPLv3 license
